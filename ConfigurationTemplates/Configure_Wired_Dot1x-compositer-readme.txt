This template is a composite template for enabling 802.1x on wired devices.  This template was created to expand the functionality of the Wired_802_1x template which was limited to IOS legacy devices.  This template expands the Wired_802_1x template to include both IOS and IOS-XE devices.

This composite template is made of 2 Templates  Enable_8021x_Globally.xml and Wired_Dot1x_IOS_XE.xml.  

This composite template will also create both of these templates in your Prime Infrastructure folder.

The Wired_Dot1x_IOS_XE.xml template is a port based template.
